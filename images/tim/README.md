## Kako da ih smestiš

Fotografije su već preuzete u tvoj **Downloads** folder preko browsera
(sa live happymedia.rs).

Prevuci ih ovde:

`C:\Claude OS\Happy Media\Happy Media sajt\images\tim\`

Nakon toga pokreni `push-sajt.bat` — tim sekcija ide live.

## Format

- 500x500 PNG (kvadrat)
- Grayscale → color na hover (podešeno u CSS-u)

Ako menjaš fotografije, zadrži identične nazive fajlova (lowercase, bez dijakritika).
